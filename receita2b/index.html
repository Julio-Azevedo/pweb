Estrutural

  • <html>

    Define o documento HTML.

  • <head>

    Contém metadados sobre o documento.

  • <title>

    Define o título do documento.

  • <base>

    Define a URL base para URLs relativas.

  • <link>

    Usado para vincular folhas de estilo externas.

  • <meta>

    Fornece metadados sobre o documento.

  • <style>

    Define estilos CSS dentro do documento.

  • <body>

    Contém o conteúdo do documento.

Agrupamento de Conteúdo

  • <header>

    Representa um cabeçalho para uma seção ou página.

  • <nav>

    Define uma seção de navegação.

  • <section>

    Representa uma seção genérica de um documento.

  • <article>

    Representa um conteúdo independente que pode ser distribuído.

  • <aside>

    Define um conteúdo relacionado, como uma barra lateral.

  • <footer>

    Representa um rodapé para uma seção ou página.

  • <h1> a <h6>

    Definem títulos de diferentes níveis.

  • <p>

    Representa um parágrafo.

  • <hr>

    Insere uma linha horizontal.

  • <pre>

    Representa texto pré-formatado.

  • <blockquote>

    Define uma citação de um bloco.

  • <ol>

    Define uma lista ordenada.

  • <ul>

    Define uma lista não ordenada.

  • <li>

    Define um item de lista.

  • <dl>

    Define uma lista de descrição.

  • <dt>

    Define um termo em uma lista de descrição.

  • <dd>

    Define a descrição de um termo.

Texto

  • <a>

    Define um hyperlink.

  • <em>

    Define texto enfatizado.

  • <strong>

    Define texto importante.

  • <small>

    Define texto menor.

  • <s>

    Define texto que não é mais relevante.

  • <cite>

    Define uma citação.

  • <q>

    Define uma citação em linha.

  • <dfn>

    Define um termo que está sendo definido.

  • <abbr>

    Define uma abreviação.

  • <time>

    Representa uma data e hora.

  • <code>

    Define um trecho de código.

  • <var>

    Define uma variável.

  • <mark>

    Define texto destacado.

  • <ruby>

    Define anotações de ruby para caracteres de texto.

  • <ruby>

    Define um texto ruby.

  • <rt>

    Define uma anotação ruby.

  • <rp>

    Define parênteses para a anotação ruby.

  • <bdi>

    Isola uma parte do texto que deve ser exibida com uma direção de texto diferente.

  • <bdo>

    Sobrepõe a direção do texto.

  • <span>

    Define uma parte em linha de um documento.

Mídia

  • <img>

    Define uma imagem.

  • <audio>

    Define conteúdo de áudio.

  • <video>

    Define conteúdo de vídeo.

  • <source>

    Define múltiplas fontes para elementos de mídia.

  • <track>

    Define faixas de texto para elementos de mídia.

  • <map>

    Define um mapa de imagem.

  • <area>

    Define uma área dentro de um mapa de imagem.

  • <picture>

    Define um container para diferentes fontes de imagens.

Formulários

  • <form>

    Define um formulário HTML.

  • <input>

    Define um campo de entrada.

  • <textarea>

    Define uma área de texto para entrada de dados.

  • <button>

    Define um botão.

  • <select>

    Define uma lista suspensa.

  • <label>

    Define um rótulo para um elemento.

  • <fieldset>

    Agrupa elementos relacionados em um formulário.

  • <legend>

    Define um título para um <fieldset>.

  • <datalist>

    Define uma lista de opções pré-definidas para um campo de entrada.

  • <output>

    Define um resultado de um cálculo.

Interativo

  • <details>

    Define um controle que o usuário pode abrir ou fechar.

  • <summary>

    Define um resumo visível para um elemento <details>.

  • <dialog>

    Define uma caixa de diálogo ou janela modal.

Semântica

  • <main>

    Representa o conteúdo principal do documento.

  • <figure>

    Representa conteúdo ilustrativo.

  • <figcaption>

    Define uma legenda para um <figure>.
